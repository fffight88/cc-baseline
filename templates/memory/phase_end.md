name: Phase End Checklist
description: Phase 종료 시 반드시 수행할 체크리스트
type: feedback
---

Phase가 끝나기 전 아래 루프를 모든 task가 완료될 때까지 반복할 것:

**[반복 루프 — 모든 task 완료 확인될 때까지 계속]**

1. 본 phase의 task 목록을 TaskList 또는 플랜에서 확인한다.
2. 완료되지 않은 task가 있는지 체크한다.
3. 미완료 task가 있으면 즉시 수행한다.
4. 수행 후 다시 1번으로 돌아가 전체 task 완료 여부를 재확인한다.
5. 모든 task가 완료 상태로 확인되면 루프를 종료하고 phase를 마친다.

**Why:** phase 종료 시 누락된 task 없이 완전히 마무리하기 위함.
