---
name: 스킬 description 예산 관리
description: 새 스킬 생성 시 description을 트리거 판단에 필요한 1~2줄로 간결하게 유지
type: feedback
---

새로운 스킬을 생성할 때 description은 간결하게 유지할 것.

**Why:** 모든 스킬 description은 세션 시작 시 컨텍스트에 로드되어 예산을 소모함. 장황한 설명은 다른 작업 영역(참조 문서, 대화 히스토리)을 줄임.

**How to apply:**
- ✅ DO: 트리거 판단에 필요한 키워드/조건만 포함하여 1~2줄로 작성
- ❌ DON'T: 사용 예시, 상세 동작 원리, 배경 설명을 description에 장황하게 기재 금지 (이런 내용은 스킬 본문에 둘 것)
