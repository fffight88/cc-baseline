description: Phase 시작 시 반드시 수행할 체크리스트
type: feedback
---

Phase가 시작될 때 아래 순서대로 반드시 수행할 것:

1. **가이드 문서 검색 및 로드**
   - 본 phase의 주제(백엔드, 프론트엔드, 디자인, 테스트, 검증, 계획 등)를 파악한다.
   - `{{HOME}}/.claude/memory/` 및 프로젝트 `.claude/` 디렉토리에서 해당 주제와 관련된 가이드 문서가 있는지 검색한다.
   - 관련 문서가 있으면 반드시 읽고 내용을 숙지한 후 작업을 시작한다.

2. **Task 목록 재확인**
   - 본 phase에 해당하는 task 목록을 TaskList 또는 플랜에서 다시 확인한다.
   - 각 task의 범위와 완료 기준을 명확히 인지한 후 작업을 시작한다.
